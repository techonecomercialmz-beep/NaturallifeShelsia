# Naturallife
NaturaLife — Website de Produtos Naturais

Este repositório contém o código-fonte do site NaturaLife, um website informativo com páginas de apresentação, produtos, sobre nós e contacto.
O projeto foi desenvolvido utilizando HTML5, CSS3, JavaScript e hospedado gratuitamente através do GitHub Pages.

Estrutura do Projeto

O projeto é composto por páginas HTML independentes:

•	index.html — Página inicial

•	produtos.html — Lista de produtos

•	sobre.html — Informações sobre a empresa

•	contacto.html — Página de contacto com formulário

Arquivos adicionais:

•	Imagens (carregadas por URL externa)

•	Fontes e ícones (Google Fonts e Font Awesome)

Tecnologias Utilizadas

•	HTML5 para estrutura das páginas

•	CSS3 para estilos e responsividade

•	Font Awesome para ícones

•	Google Fonts (Poppins)

•	JavaScript para menu responsivo

•	GitHub Pages para hospedagem

 Como o Projeto Foi Criado
 
1.	Criação das páginas HTML
   
o	Cada página foi estruturada seguindo boas práticas do HTML5.

o	O cabeçalho foi desenvolvido com navegação responsiva utilizando flex, ícones e menu mobile.

o	Foi criado um banner com imagem de fundo e textos sobrepostos.

o	A página contacto.html inclui um formulário com campos validados (required).

3.	Aplicação de estilos (CSS)
   
o	O layout utiliza a fonte Poppins importada via Google Fonts.
o	A paleta de cores segue o tom verde escuro (#2f553a).
o	Criou-se responsividade com media queries para telas menores.
o	Adicionou-se sombras, bordas arredondadas e espaçamentos modernos.

5.	Funcionalidade do menu mobile
   
o	Implementada com JavaScript simples:
	O botão hamburguer abre/fecha o menu.
	O menu aparece apenas em dispositivos com largura < 780px.

7.	Criação da página de contacto
   
o	Formulário funcional com campos: nome, email, assunto e mensagem.
o	Ação configurada para contacto.php (somente funciona em servidores com PHP).

Como Testar o Projeto Localmente

1️⃣ Clonar o repositório

git clone https://github.com/AnaTovele/NaturallifeShelsia
NaturallifeShelsia.git

2️⃣ Abrir os arquivos no navegador

Você pode simplesmente clicar duas vezes em qualquer arquivo .html (ex.: index.html) e ele abrirá no navegador.

3️⃣ Testar o menu mobile

•	Reduza o tamanho da janela do navegador
•	Verifique se o botão hamburguer aparece
•	Clique para abrir/fechar o menu

4️⃣ Testar o formulário

O formulário enviará dados para contacto.php caso esteja num servidor com suporte a PHP.
No GitHub Pages este recurso não funciona, pois o GitHub não executa PHP.

Publicação do Projeto (Hospedagem no GitHub Pages)

1️⃣ Criar o repositório no GitHub

•	Nome do repositório: naturalife-site (ou qualquer outro)

2️⃣ Enviar os arquivos

Envie todos os arquivos HTML, CSS (se tiver externo), imagens e scripts.

3️⃣ Ativar o GitHub Pages

1.	Vá para Settings
2.	Vá para Pages
3.	Em Build and Deployment, escolha:
o	Source: Deploy from branch
o	Branch: main → /root
4.	Clique em Save
   
4️⃣ Aguardar a publicação

Após alguns segundos, o GitHub exibirá o link:
https://seuusuario.github.io/naturalife-site/

5️⃣ Testar o site hospedado

Acesse o link e confirme:
•	Navegação entre páginas
•	Menu mobile
•	Carregamento das imagens
